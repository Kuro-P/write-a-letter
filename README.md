# write-a-letter
canvas 绘制触屏写字编辑器（慕课网练习）

## 主要模块
1. canvas 坐标轴的转换
2. 消除笔画锯齿，光滑笔画
3. 根据用户运笔速度对笔画粗细的调整
4. 移动端 touch 相关事件与 PC 端鼠标事件之间兼容

## 开发小记
* 需要阻止事件默认行为，防止浏览器页面滚动
